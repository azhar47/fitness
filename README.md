# 🏋️‍♂️ Fitness Website

This is a **Fitness Website Template** collected from [ThemeWagon](https://themewagon.com/themes/fitness-template/). I am using this template for **educational and research purposes** as part of my **Master’s thesis on website speed optimization**.

---

## 📌 **About the Project**
This project is a **static website** built using **HTML, CSS, and JavaScript**. It provides a responsive and visually appealing layout designed for fitness and gym-related websites.

This template was originally developed by **ThemeWagon**, and I have utilized it in my **Master's thesis titled "Website Speed Optimization: A Comparative Study on Static, Dynamic, and WordPress CMS Websites."** 

---

## 📈 **Purpose of the Repository**
The objective of this repository is to analyze and optimize the speed performance of a **static website** in comparison to dynamic and WordPress-based websites.

### **Key Research Areas:**
- Page load time optimization
- Caching strategies for static websites
- Image compression techniques
- Minification of assets (CSS, JS, HTML)
- Performance comparison with dynamic and CMS-based websites
